# nlp220_hw3

Multilabel Classification of arXiv Paper Summarizations

## Best model

Logistic Regression

## Best Features

Combine Title and Summaries columns, and fit with CountVectorizer embedding, setting n-gram to (1,2) and max_features to 99999.

## Results

Micro-F1: 0.8143\
Macro-F1: 0.4367
